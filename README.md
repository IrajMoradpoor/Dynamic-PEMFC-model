# Dynamic PEM Fuel Cell Model (PEMFC)

This repository contains a physics-based Python model of a Proton Exchange Membrane Fuel Cell (PEMFC) operating under dynamic load conditions.

## Model Features

- Electrochemical voltage modeling
- Nernst potential calculation
- Activation, ohmic, and concentration losses
- Variable current density operation
- Stack power calculation

## Outputs

The model simulates:

- Cell voltage
- Power output
- Performance behavior under part-load operation

## Technologies

- Python
- NumPy
- Pandas

## Research Context

This model was developed as part of research on dynamic hydrogen energy systems and flexible energy operation.
